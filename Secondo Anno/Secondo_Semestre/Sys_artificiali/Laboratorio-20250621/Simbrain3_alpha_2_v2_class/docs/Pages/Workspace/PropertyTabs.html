Property Tabs


In the bottom of the simbrain desktop is a set of tabs that can be used to monitor the state of the workspace and set properties.

Components

A list of the components in the workspace (see: components for more information). 

In List:

Type: Shows the kind of workspace components (network, bar chart, time series, etc).

GuiOn: A simple checkbox allowing the user to enable (default) or disable the component's GUI. Doing this will visually remove the component from the workspace, lowering the computational time and memory usage of all network operations.

Update On: A simple checkbox allowing the user to enable (default) or disable the updating of a network component. If this option is not selected, the component will not update when the global (workspace) update is activated (see: Update for more information).

Priority: By double clicking on the number under the priority heading, the option appears to change this number. This allows the user to change the priority of a workspace component relative to other workspace components during global updates. The lower the number, the higher the component's update priority.


Couplings

A list of all the active couplings in the workspace listing their parent component and the direction of information flow (see: Couplings for more information).

Terminal

A beanshell terminal allowing the user a command line level interface with Simbrain.  See the discussion in scripting.

Updator

Status of updator. See update.
