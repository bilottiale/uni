Simbrain > Workspace > Menus

Menus

File

Open Workspace: Opens a workspace from a stored .zip file.  That is, open a configuration of workspace components (networks, worlds, plots, etc.).

Save Workspace: Save the current workspace as a .zip file.  All components and couplings are saved.

Save Workspace As: Save the current workspace and specify a name for the file.

Clear Workspace: Remove all components and couplings from current workspace, and remove any additional windows from the desktop.

Open Network: Open a Network in a new network frame.

Open World > OdorWorld: Open an OdorWorld in a new world frame.

Open World > DataWorld: Open a DataWorld in a new world frame.

Workspace Properties: Open workspace preferences dialog.

Insert

New Network: Insert a new Network component in to the workspace.

New Plot: Insert a new Plot component (of a specified type) in to the workspace.

New Odor World: Insert a new OdorWorld

New DataWorld: creates a new DataWorld.

Console: Create a new BeanShell console. 

Help

Help: calls up this help page.
