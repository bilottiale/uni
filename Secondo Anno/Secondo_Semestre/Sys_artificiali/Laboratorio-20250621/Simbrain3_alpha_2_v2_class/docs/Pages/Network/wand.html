Wand tool

Wand Tool:

The wand tool, which can be found on the network toolbar, is a tool used to alter the activations of neurons within its range.

As of the current version of Simbrain the wand only injects activation.  Other uses for the wand tool are planned for future releases.

To use the wand:
    1) Click on the wand icon
    2) Pass the wand over neurons while holding the mouse key
    3) All neurons will reach their maximum level (the “upper bound” value), as shown below.


The radius in pixels of the wand's effect can be adjusted by going to Edit > Wand/Edit Mode Preferences within a network window.
