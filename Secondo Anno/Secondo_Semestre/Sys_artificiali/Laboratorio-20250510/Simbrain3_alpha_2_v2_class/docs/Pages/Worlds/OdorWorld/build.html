Creating worlds

To change a world the user can either manually edit a world's xml file (these files should be self-explanatory), or follow these steps:

1. Create new objects using the pop up menu in odor world.

2. Set the new objects properties using the entity dialog box.

3. Establish the desired couplings.

4. When a world has been configured, save it using the "Save" or "Save as" command in the world's File menu.


The World will be affected as the coupled networks are iterated. If a neuron coupled to an agent's movement commands, the agent will only move when the coupled network is iterated.
