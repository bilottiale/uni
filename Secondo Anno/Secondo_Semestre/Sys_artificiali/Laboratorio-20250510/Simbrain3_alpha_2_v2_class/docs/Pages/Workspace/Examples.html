Simbrain > Workspace > Examples

Examples

Open an existing workspace using File > Open workspace, move and resize the windows slightly. This workspace can be saved using using File > Save workspace.

Select File > Clear workspace to remove all current components from the workspace, open a single network using File> Open network, open a sample network, and open a gauge using the gauge button in the network frame. Run the network using its play button and observe the pattern which results in the gauge.
