Clamped Synapse

A Clamped Synapse does not changes its value other than by manual adjustment or an external subnetwork (e.g. an LMS network). Utilizing clamped synapses can be essential when testing or training a network. For example, once a network has learned what you want it to learn, its synapses can be clamped in order to assure consistent behavior.

Synapses can also be clamped without setting their synapse-type specifically to Clamped. This can be useful when one requires specially tuned synapses and only wants to temporarily clamp them. To do this, use the clamp weights command: Network > Edit > Clamp weights.
