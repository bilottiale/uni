◆ analyze_string()

int analyze_string ( const char * str,
StringInfo * info )

Analizza una stringa e restituisce informazioni dettagliate.

Questa funzione esamina una stringa di input e calcola varie statistiche, memorizzandole in una struttura StringInfo.

Parametri
[in]strLa stringa da analizzare (non può essere NULL)
[out]infoPuntatore alla struttura dove memorizzare i risultati
Restituisce
0 in caso di successo, -1 in caso di errore
Valori di ritorno
0Analisi completata con successo
-1Parametri non validi (str o info sono NULL)
Esempio di utilizzo:
StringInfo info;
char text[] = "Hello World";
if (analyze_string(text, &info) == 0) {
printf("Lunghezza: %zu\n", info.length);
printf("Parole: %zu\n", info.word_count);
}
