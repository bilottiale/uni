Descrizione dettagliata

Rappresenta un punto in uno spazio bidimensionale.

Questa struttura incapsula le coordinate cartesiane di un punto nel piano bidimensionale, fornendo un'interfaccia semplice per operazioni geometriche di base.

Nota
Tutte le coordinate sono in formato floating-point
Si veda anche
Point3D per punti tridimensionali
