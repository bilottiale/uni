◆ factorial()

unsigned long factorial ( int n)

Calcola il fattoriale di un numero.

Questa funzione calcola il fattoriale di un numero intero positivo utilizzando un approccio iterativo per evitare overflow dello stack.

Parametri
nIl numero di cui calcolare il fattoriale (deve essere >= 0)
Restituisce
Il fattoriale di n, oppure 1 se n è 0 o negativo
Avvertimento
Per valori di n > 20, il risultato potrebbe andare in overflow
Nota
La complessità temporale è O(n), quella spaziale è O(1)
unsigned long result = factorial(5); // result = 120
printf("5! = %lu\n", result);

◆ power()

double power ( double base,
int exponent )

Calcola la potenza di un numero.

Calcola base^esponente utilizzando l'algoritmo di esponenziazione veloce (exponentiation by squaring) per ottimizzare le prestazioni.

Parametri
baseLa base dell'operazione di potenza
exponentL'esponente (deve essere >= 0)
Restituisce
Il risultato di base^esponente
Precondizione
exponent >= 0
Postcondizione
Il risultato è sempre positivo se base > 0
Nota
Complessità: O(log n) dove n è l'esponente
Esempio:
double result = power(2.0, 10); // result = 1024.0
